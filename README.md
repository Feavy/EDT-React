# FlopEDT WebApp
Système de visualisation, codé en ReactJS, de l'emploi du temps généré par le logiciel [FlopEdt](http://www.flopedt.org/).

Implémentation : [https://feavy.netlify.com/edt/](https://feavy.netlify.com/edt/).
